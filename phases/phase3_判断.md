# Phase 3: 判断 — 判断ログ & 失敗台帳

> 「なぜそう決めたか」と「何を間違えたか」を記録し始める。

## チェックリスト
- [ ] `knowledge/decisions/` フォルダを作成した
- [ ] 判断ログを3件以上書いた
- [ ] `templates/failure_log.md` を元に失敗台帳を作成した
- [ ] 失敗を3件以上記録した
- [ ] 業種固有の知識ファイルを1つ以上作った

## 判断ログのフォーマット
```
## [タイトル]
- 対象: 何について
- 判断: 何を選んだか
- 理由: なぜ
- 却下案: 選ばなかった選択肢と理由
- 次回ルール: 同じ状況でどうするか
```

## 失敗台帳のフォーマット
```
### [番号] [タイトル]
- 日時: YYYY-MM-DD
- 誰が: エージェント名
- 何を: 何が起きたか
- なぜ: 根本原因
- 学び: 何を学んだか
- 対策: 今後どうするか
```

## 次へ進む条件
判断層と失敗層が2週間以上運用されたら Phase 4 へ。
