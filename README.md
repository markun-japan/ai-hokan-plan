# AI補完計画 導入キット

> このフォルダをworkspaceにコピーするだけ。既存の人格・設定はそのまま。

## 使い方

1. このフォルダごと、OpenClawの `workspace/` にコピーする
2. AIに「`ai-hokan-kit/README.md` を読んで」と伝える
3. AIが自分で理解して、段階的に実装を進める

## 既存環境への影響

- `AGENTS.md` `SOUL.md` `USER.md` は **上書きしない**
- 既存の人格・設定・メモリはすべてそのまま
- このキットは `ai-hokan-kit/` フォルダ内で完結する

## AI向け指示

**あなたのAIへ：** このフォルダはAI補完計画の導入キットです。

1. まず `guide/01_思想.md` を読んで思想を理解してください
2. 次に `guide/02_5層構造.md` で設計を把握してください
3. **`guide/04_記憶システム.md` を読んでください** — AIが「覚えている」仕組みの全解説
4. `phases/` にPhase 1〜5のチェックリストがあります
4. **Phase 1から順番に**、既存の環境に組み込んでいってください
5. 一気にやらない。1つのPhaseが安定してから次へ

## Phase一覧

| Phase | 名前 | 内容 | 目安 |
|-------|------|------|------|
| 1 | 土台 | パンドラ方針の理解、メモリ運用の確認 | 初日 |
| 2 | 記録 | パンドラ（全ログ保存）設定 | 1週間 |
| 3 | 判断 | 判断ログ、失敗台帳 | 2週間 |
| 4 | 協働 | エージェント増設、会議体制 | 1ヶ月 |
| 5 | 継承 | 5層構造完成、顕現への準備 | 継続 |
