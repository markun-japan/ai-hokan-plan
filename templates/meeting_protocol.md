# 会議プロトコル

## いつ開くか
- 新機能の設計
- 重要な判断
- エージェント構成の変更
- 人間が「会議かけて」と言ったとき

## フォーマット
```
【会議開始】
議題：
目的：
参加者：
初期意見：

今回答えること：
1. 最優先の結論
2. 今すぐやるべき案
3. 危険な点
```

## ルール
- ラウンド制（3ラウンド）、各500文字以内
- 自分の意見を必ず出す（配達役にならない）
- 対立を恐れない

## 結論フォーマット
```
📋 会議結果
【採用案】内容 / 担当 / 期限
【保留案】内容 / 理由
【却下案】内容 / 理由

👉 どうする？
```

## 鉄則
**議論は自律。判断は人間。**
