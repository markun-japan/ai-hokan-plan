# Memory Candidates（記憶候補置き場）

> 日次ログから拾った「長期記憶に上げるかもしれない」項目の仮置き場。
> 毎週の棚卸しで承認されたものだけ MEMORY.md に昇格する。
> これによりMEMORY.mdの肥大化を構造的に防止する。

## 候補リスト

### [日付] [タイトル]
- **重要度:** 高 / 中 / 低
- **信頼度:** 確定 / 要確認 / 推測
- **公開範囲:** 全エージェント / 管理者のみ / 本人のみ
- **期限:** なし / YYYY-MM-DD
- **出典:** memory/YYYY-MM-DD.md / 会議結果 / 外部情報
- **内容:** （1-2行で要約）

---

## 昇格ルール
1. 週次棚卸し（金曜）で候補を全件レビュー
2. 「確定」かつ「中以上」→ MEMORY.md に転記
3. 「期限切れ」→ 削除
4. 「要確認」→ 確認後に再判定
5. 転記したら候補リストから削除

## アンチパターン
- ❌ 候補を溜めすぎる（20件超えたら強制棚卸し）
- ❌ 候補を直接MEMORY.mdに書く（パイプラインを飛ばさない）
- ❌ 機密情報を候補に書く（公開範囲を必ず設定）
